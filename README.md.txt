# 🌾 KisanConnect

Smart Seasonal Farm Labor & Machinery Matching System

## Problem
Farmers face labor shortages and machinery unavailability during seasonal farming operations.

## Solution
KisanConnect connects farmers, laborers, and machinery owners using AI-based smart matching.

## Features
- Labor booking system
- Machinery rental
- AI demand prediction
- Location-based matching
- Fair pricing recommendations

## Tech Stack
- Flutter (Frontend)
- Node.js + Express (Backend)
- Python (AI Model)
- PostgreSQL Database

## Impact
Improves rural employment, reduces crop loss, and ensures fair pricing.
